# 重复文件查找
一个`.NET 6 WPF`写的重复文件查找工具，解放你的硬盘空间，红盘用户的福音。  
![FindDuplicateFiles.png](https://i.loli.net/2021/07/24/XiqF6fy8xlrpzwc.png)  

# 说明
编译时需要在`Updates`文件夹下添加一个`Update`文件（自动更新的接口地址）  
```json
{
  "Url": "xxxxxxx"
}
```
