## 使用说明  
* 选项 - 启用图片文件预览  
启用时，在搜索结果中点击图片文件，右下角会出现图片预览结果。  

* 选项 - 删除时移除空目录  
删除文件的时候，如果文件夹中只剩余当前文件，删除的时候会顺便删除文件夹。  

* 结果区域 - 一键选中重复文件  
自动选中所有重复文件（每个文件只保留一份）。  

* 结果区域 - 自定义选择重复文件  
弹出筛选窗口，用户可以根据查找路径选择本次要删除的目标文件夹。

* 其它  
其它选项都比较通俗易懂，还请同学自行摸索。
